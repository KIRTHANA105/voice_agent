# Assets

Static assets for documentation and submission.

## Folders

| Folder | Purpose |
|--------|---------|
| `screenshots/` | Application screenshots for reports and presentations |
| `diagrams/` | Architecture and flow diagrams |

## Adding Screenshots

Capture the running app at [http://localhost:8501](http://localhost:8501) and save PNG files here, for example:

- `screenshots/home.png` — main chat screen
- `screenshots/voice-input.png` — voice interaction
- `screenshots/sample-response.png` — example Q&A

> **Note:** Screenshot files are not included in the repository by default. Add them before final presentation if required by your institution.
